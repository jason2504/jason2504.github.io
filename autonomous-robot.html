Mobile Robot Platform

Autonomous Competition Robot with Web Interface Control

Project Overview

This MEAM 5100 Design of Mechatronic Systems project involved designing, building, and programming a complete autonomous mobile robot platform for a League of Legends-inspired competition. The robot successfully demonstrated wall-following capabilities, autonomous attack functionality, manual control via web interface, and robust mechatronic integration.

Competition Strategy

Our team developed a comprehensive strategy utilizing both autonomous and manual control modes. During autonomous periods, the robot followed walls using ultrasonic sensors and PD control, automatically attacking the nexus and ramp tower when reaching calibrated Vive coordinates. The manual control phase employed a custom web interface with WASD keyboard controls for precise human-operated movement and strategic gameplay.

Technical Implementation

The robot featured a 3-layer mechanical design housing all necessary components:

  • Autonomous Navigation: HC-SR04 ultrasonic sensors with PD control for wall following
  • Positioning System: Vive sensors for precise localization and autonomous attack coordination
  • Attack Mechanism: MG995 servo-driven attack arm with continuous 90-degree sweep motion
  • Communication: I2C communication with TopHat for health monitoring and game state management
  • Manual Control: Real-time web interface with WASD controls and visual feedback

Performance Results

The robot achieved a perfect evaluation score of 50/50 points, successfully completing:

  • Full wall-following circuit with robust PD control
  • Autonomous attack on nexus target
  • Manual vehicle control with responsive web interface
  • Reliable I2C communication and health monitoring
  • Successful ramp navigation and stationary target engagement

System Architecture

Mechanical Design

  • 3-layer acrylic platform design
  • Parallax Feedback 360° servo motors
  • Custom laser-cut drive wheels with rubber bands
  • Attack arm with MG995 servo motor
  • Protective bumper system

Control Systems

  • PD controller for wall following (Kp=0.4, Kd=0.02)
  • Real-time web interface with JavaScript
  • Autonomous mode switching via push buttons
  • Health monitoring and game state management

My Contributions

As the mechanical design lead, I was responsible for the complete physical design and assembly of the robot platform:

  • Mechanical Design: Designed the entire 3-layer robot chassis using SolidWorks, including motor mounts, sensor housings, and attack arm mechanism
  • Manufacturing: Managed laser cutting, 3D printing, and assembly processes, creating custom drive wheels and protective bumper systems
  • Motor Control: Implemented manual motor control code and attack arm servo logic for competition gameplay
  • I2C Communication: Developed communication protocols with TopHat for health monitoring and game state management
  • System Integration: Led final assembly and testing, ensuring reliable performance across all mechanical and electrical subsystems
  • Competition Preparation: Conducted extensive testing and calibration, achieving the perfect evaluation score through systematic debugging and optimization

Key Learnings

This project provided valuable insights into mechatronic system design and real-world robotics challenges:

  • Robust Design: Learned the importance of mechanical redundancy and protective features for competition environments where robot-to-robot contact is expected
  • System Integration: Gained experience in coordinating mechanical, electrical, and software subsystems for reliable autonomous operation
  • PD Control: Developed practical understanding of control theory implementation, particularly for sensor-based navigation systems
  • Debugging Methodology: Established systematic approaches to hardware debugging, especially for sensitive components like ultrasonic sensors
  • Competition Strategy: Learned to balance autonomous reliability with manual control flexibility for optimal performance in dynamic competition scenarios
