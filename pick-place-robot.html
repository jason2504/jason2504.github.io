Pick-and-Place Robot

Vision-Aided Pick-and-Stack Competition with 7-DOF Manipulator

Project Overview

This MEAM 5200 Introduction to Robotics final project challenged teams to develop autonomous pick-and-place capabilities using the Franka Emika Panda 7-DOF manipulator. Our team "Robust Robotics" achieved 4th place in the competitive challenge by implementing advanced computer vision, inverse kinematics, and trajectory planning for precision block manipulation in a dynamic environment.

Competition Challenge

The competition required robots to pick wooden cubes from both static platforms and a rotating turntable, then stack them to maximize points. Static blocks were worth 10 points per millimeter of altitude, while dynamic blocks on the rotating platform were worth 20 points per millimeter, creating a strategic challenge that balanced risk and reward.

Technical Approach

Our solution integrated multiple sophisticated robotics techniques:

  • Computer Vision: AprilTag detection system using Intel RealSense D435 camera for real-time block localization and pose estimation
  • Motion Planning: Precomputed waypoint approach with safe_move_to_position() for collision-free trajectories
  • Dynamic Prediction: Time-based reasoning algorithm to predict future positions of moving blocks with 5-second calibration offset
  • Inverse Kinematics: Gradient descent optimization with pseudo-inverse and joint centering for reliable manipulation

Strategic Innovation

Our team developed a unique two-phase stacking strategy to maximize points while ensuring reliability:

  • Phase 1: Create separate stacks of 4 dynamic blocks and 4 static blocks
  • Phase 2: Combine stacks by moving dynamic blocks on top of static stack
  • Final result: 8-block tower worth 28,000 points (theoretical maximum)
  • Failsafe system: Camera verification after each placement to ensure successful pickup

Technical Implementation

Vision System

  • AprilTag detection with camera calibration correction
  • Homogeneous transformation from camera to world frame
  • Static and dynamic offset calibration for different block types
  • Optimal detection heights: 0.5m static, 0.45m dynamic

Motion Control

  • IK_position_null with joint centering secondary task
  • Gradient descent with pseudo-inverse optimization
  • 0.001 linear and angular tolerance for precision
  • Arm speed optimization at 0.2 for stability

Dynamic Block Prediction Algorithm

For moving blocks on the rotating turntable, we developed a sophisticated prediction system:

  • Motion Analysis: Turntable rotation at 0.4 RPM with known radius (0.305m)
  • Time Calculation: End effector travel time based on Euclidean distance and 0.3 m/s velocity
  • Position Prediction: Future block pose using rotation matrices and coordinate transformations
  • Gripper Alignment: Strategic orientation to allow blocks to "slide" into gripper channel

Results and Performance

Simulation Results

  • 100% success rate for static block stacking (3 tests)
  • 100% success rate for dynamic block stacking (3 tests)
  • Consistent performance across red and blue team configurations
  • Average completion times: 2:46 static, 3:25 dynamic, 2:36 combining

Competition Performance

  • Successfully achieved 4th place out of all competing teams
  • Maximum score: 8,000 points with 4-block dynamic stack
  • Consistent performance in preliminary rounds
  • Challenges in finals due to opponent interference strategies

Challenges and Lessons Learned

This project provided valuable insights into real-world robotics challenges:

  • Simulation vs Reality Gap: Learned the importance of extensive hardware testing, as camera offsets and timing differed significantly from simulation
  • Dynamic Environment Adaptation: Discovered how opponent strategies can affect system performance, highlighting the need for robust failure recovery
  • Vision System Reliability: Gained experience with AprilTag detection challenges including lighting conditions, reflections, and motion blur
  • Competition Strategy: Learned to balance ambitious goals with reliable execution, as our conservative approach secured consistent performance
  • Team Coordination: Developed skills in collaborative robotics development, integrating vision, planning, and control subsystems effectively
